# が
## 第一种用法
解释：
- 格助词，表示动作，存在，情况的主体
- 谁／什么做了这个动作
- 谁／什么具有这个状态或性质
- 纯粹描述现象发生

结构：
名词/名词的词语 +  が + 通常是跟います・あります

例句：<br>
1. 公園で子供たちが遊んでいます
2. あそびに、コンビニがあります
3. 机の上に本があります
4. 空が青いなぁ
5. 事故が起きました

## 第二种用法

解释：
- 格助词，表示希望，好恶，能力等的对象
- 【想要】 的东西或 【想做】的事情
- 【喜欢】 或讨厌的对象
- 【会，懂，能做到】的事情

结构：
名词/名词的词语 +  が

例句：<br>
1. 水がのみたい
2. 父は納豆が嫌い
3. 日本語が話せますか？
4. ちょっといみがわからない
